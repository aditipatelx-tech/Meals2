# Meal Roulette

A polished modular recipe picker with search, filters, modal details, and dark mode.

## File structure

- `index.html`
- `styles.css`
- `app.js`
- `data/recipes.json`
- `data/categories.json`
- `components/card.js`
- `components/modal.js`
- `components/filters.js`
- `utils/dom.js`
- `utils/storage.js`
- `README.md`

## Notes

This build uses native ES modules in the browser, so make sure the files are served from a web server or GitHub Pages rather than opened directly as a local file.

## Setup

1. Upload all files to the same repository.
2. Keep the folder names exactly as shown.
3. Make `index.html` the entry page.
4. Deploy with GitHub Pages.

## What it includes

- Random meal picker
- Search by name, cuisine, ingredient, or instruction text
- Cuisine and diet filters
- Full recipe modal
- Dark mode with saved preference
- Clean modular file organization
